def standardize_data(data):
    
    col_names = data.columns
    # calls standardization function z = (x - u) / s
    scaler = preprocessing.StandardScaler()
    # fits data and transforms
    standardized_df = scaler.fit_transform(data)
    # convert to dataframe with column names
    standardized_org_data = pd.DataFrame(standardized_df, columns=col_names)
    return standardized_org_data

def dimensionality_reduction(stzd_data,cmpnts):
    pca = PCA(n_components=cmpnts)
    pca_components = pca.fit_transform(stzd_data)
    pca_org_data = pd.DataFrame(pca_components, columns=['PC1','PC2'])
    return pca, pca_org_data

7. Optimal number of clusters (k)¶

7.1 Elbow method¶

In [63]:
#Identify optimal clusters(k)

#Elbow method
score = []

# We begin with a minimum number of clusters of atleast 2
K = range(2,11)

for k in K:
    kmeanModel = KMeans(n_clusters=k)
    kmeanModel.fit(trans_data)
    # Inertia: It is the sum of squared distances of samples to their closest cluster center.
    score.append(kmeanModel.inertia_)

# Displays the sum of squared distance to its closest cluster.
#print("Inertia Score (α)\n")
#for x in score:
#    print(round(x,2))
#print("\n")

plt.figure(figsize=(16,8))
plt.plot(K, score, 'bx-')
plt.xlabel('No of clusters (k)')
plt.ylabel('Inertia Score (α)')
plt.title('Elbow method to determine optimal (k)')
plt.show()

The elbow method helps to choose the optimum value of k (number of clusters) by fitting the model with a range of values of k. Let us start by understanding the cost function of K-means clustering. We define cost function of K-means clustering as 'Inertia Score (α)' which is sum of squares of distance of data point to their closest cluster. We can see that the cost function decreases with number of iterations/clusters (k).

The above shows figure shows a plot of No of clusters (k) vs Inertia Score (α).

Therefore, we determine that elbow formation takes place at k=5(optimal), although it is not a very sharp elbow formation.

7.2 Silhouette method¶

In [64]:
sil = []
K = range(2,11)

# We begin with a minimum number of clusters of atleast 2
for k in K:
    kmeans = KMeans(n_clusters = k).fit(trans_data)
    # Predict the closest cluster each sample in X belongs to.
    predict = kmeans.predict(trans_data)
    sil.append(silhouette_score(trans_data,predict))

# Displays the silhouette score
#print("Silhouette Score (β)\n")
#for x in sil:
#    print(round(x,2))
#print("\n")

plt.figure(figsize=(16,8))
plt.plot(K, sil, 'bx-')
plt.xlabel('No of clusters (k)')
plt.ylabel('Silhouette Score (β)')
plt.title('Silhouette method to determine optimal (k)')
plt.show()

The silhouette score varies between -1≤β≤1. A value close to 1 implies that the instance is close to its cluster is a part of the right cluster. Whereas, a value close to -1 means that the value is assigned to the incorrect cluster. The Silhouette coefficient is calculated using the mean intra-cluster distance x and the mean nearest-cluster distance y for each data point.

The Silhouette Coefficient is defined as (y - x) / max(x, y). Silhouette method shows a peak characteristic whereas the Elbow method shows a slight bend, appearing like the formation of hand with the elbow being the point of bend. The silhouette method reaches its global maximum at the optimal k.

Therefore, from our plot we can deduce that optimal k=2 from the Silhouette method.
